# Changelog

All notable changes to the MoodCode extension are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.0.1] - 2026-05-21

### Added

- Time-of-day mood detection with automatic VS Code theme switching
- Status bar mood indicator
- Manual mood override for 1, 2, or 4 hours
- WebSocket sync for dashboard config updates
- Mood switch logging via local backend API
- Optional auto-start of bundled backend on activation
- Commands: Set Mood Override, Refresh Mood, Open Dashboard
